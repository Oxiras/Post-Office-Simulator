Included:
- Poffice.java: Memory program of the application.
- Customer.java: Processor program of the application.
- Project 2 Design: Design document.
- Project 2 Summary: Summary of the project.

Follow these steps to run the application:
1. Place following files in the same folder:
	- Poffice.java
	- Customer.java
2. Run following commands:
	- javac Poffice.java Customer.java
	- java Poffice
